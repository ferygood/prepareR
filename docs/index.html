Install R, RStudio, and R packages

R is the programming language and RStudio is the user interface to let us use R.

If you want to explore by yourself, here are the manuals to install R and RStudio.

In brief:

Windows:

  1. Download the current R version for windows here: R download link (R for windows) and follow the instructions.

  2. Download RStudio here: RStudio download link (windows) and follow the instructions.

macOS and Linux:

  1. Download the current R version for windows here: R for macOS (M1,2..Chip) | R for macOS (Intel)| R for Ubuntu(Linux) and follow the instructions.
  2. Download RStudio here: RStudio for macOS, ,RStudio for Ubuntu and follow the instructions.

Next step is to install all pre-requested packages for the course.

Install pre-requested packages

Please type the below command in your console one by one:

# Step1. Install packages from CRAN
install.packages(c("dplyr", "Seurat", "devtools", "rmarkdown"))

# Step2. Install presto
devtools::install_github("immunogenomics/presto")

# Step3. Check if you have BiocManager
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

# Step4. Install hdf5r, DESeq2, edgeR
BiocManager::install(c("hdf5r", "DESeq2", "edgeR"))

Check everything is settle

Check you have R and all packages are available for the correct version. Try execute the code below in your console in RStudio. You should get something like the content below without error. Try to install the same or at least above versions of the packages.

library(dplyr)
library(Seurat)
library(presto)
library(hdf5r)
library(DESeq2)
library(edgeR)
